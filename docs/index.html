Travels in Europe

We will be documenting Mary Behrend's travel journals from the family's time in Europe. Specifically their time in France.

Using XML, we will be adding data to the documents and posting them here to our website.

The Navbar will contain links to this page, a page with photos/scans of each of the pages of the journal, and a link to the GitHub repo containing the whole project.
